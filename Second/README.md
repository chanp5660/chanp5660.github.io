# 2019.04.02

### 자료조사

> 취직 사이트에서 나의 이력서와 비슷한 사람(어느 정도 기준을 정할 수 있어야 한다.)들이 어느 회사에 지원하고 있는지 매일 데이터를 수집한다.(취업 공고는 매일 달라지기 때문에)
> <br></br>
이유 : 
> - 현재 사이트들의 시스템은 고용정보가 올라오면 자신의 이력서와 맞는 것들을 필터하여 추천하는 시스템이다. 단순하고 강력한 정보를 준다고 생각한다. 
>
> - 지금의 트렌드는 첫째는 속도, 둘째는 단체가 아닌 개인에 맞추어가는 트렌드이다. 원래 시스템은 자신이 사이트에서 추천해주는 것을 다 본다음 고른다. 그것보다는 자신과 비슷한 사람들이 어떤 생각을 하고 있는지 좀 더 빠르게 판단할 수 있게 될 것이다.
>
> - 따라서 우선 고객이 형식에 맞게 데이터를 입력하면 같은 이력서(어느정도)의 사람들이 어느 회사의 지원을 했는지 알 수 있는지 확인할 수 있도록 데이터를 수집한다.
>
> 이용한 사이트 : [사람인](http://api.saramin.co.kr/documents/job-search-api)  안되는 사이트 : 워크넷, 잡코리아
>
> 위의 3개의 사이트를 다 이용하고 싶었지만 개인은 할 수 없고 채용관련 기관만 이용할 수 있다고 하여 사람인만 이용한다.
