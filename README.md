## 실행 방법

1. 루트 경로에 ".env" 파일을 생성하고 아래와 같이 작성

```plaintext
SAVE_PATH_URL = 'C:\\Users\\master\\Desktop\\PARA' # 블로그 업로드 파일 저장 경로
SOURCE_PATH_URL = 'C:\\Users\\user\\Desktop\\chanp5660\\PARA' # 옵시디언 파일 저장 경로
```
2. obsidian_to_blog_sync.bat 파일 또는 obsidian_to_blog_sync.py 실행

## 목적

- 업로드 자동화를 통한 블로그 운영 효율화
